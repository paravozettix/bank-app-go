# Bank Console Application

Консольное приложение на Go, имитирующее работу банковской системы. 

### Основные возможности:
* **Управление счетами:** Создание аккаунтов с уникальными ID.
* **Финансовые операции:** Переводы между пользователями и пополнение баланса.
* **Безопасность:** Проверка наличия средств перед транзакцией и валидация ID.
* **История:** Хранение истории операций для каждого счета.

### Технический стек:
* **Go Modules:** Проект разбит на пакеты (`main` и `bank`) для лучшей структуры.
* **Data Structures:** Использование `map` для быстрого поиска счетов и `slices` для истории.
* **Pointers:** Работа с данными через указатели для изменения состояния объектов.
